Overview

CALC2000 is a simple COBOL program that calculates the future value of an investment using compound interest. The program runs the calculation three times, doubling the investment amount after each calculation.

This assignment demonstrates core COBOL concepts such as arithmetic operations, looping with PERFORM, and formatted output.

Program Details

Initial Investment Amount: 1000

Number of Years: 10

Yearly Interest Rate: 5.5%

Investment Doubling: The investment amount is doubled twice during execution
